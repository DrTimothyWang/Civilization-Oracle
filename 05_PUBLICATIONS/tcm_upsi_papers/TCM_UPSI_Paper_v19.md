# TCM-UPSI: 五运六气与历史危机预测建模研究
## 论文 v19.0 — 外部验证危机与科学诚实报告

**作者**: TCM-UPSI 研究团队  
**日期**: 2026-06-08  
**版本**: v19.0 (External Validation Crisis Edition)  
**状态**: 阶段性总结 — 包含正面发现与负面结果

## 摘要

本研究探索中医五运六气（WuYun-LiuQi）周期与全球历史危机事件的关联性，构建机器学习预测模型。经过59个研究阶段的迭代，我们取得以下**混合结果**：

**正面发现**:
- 内部数据库（1,202事件，1906-2024）上AUC=0.9538（年度）/0.9941（月度）
- Temporal特征（历史危机计数）主导预测（48.1%重要性）
- Climate特征（ENSO/PDO/全球温度）贡献+2.1% AUC提升
- 月度模型通过12×样本扩展实现AUC突破

**负面发现（科学诚实报告）**:
- **UCDP/PRIO外部验证失败**: AUC=0.5547（几乎随机）
- **WuYun统计检验不显著**: 所有chi-square p>0.05
- **事件定义不匹配**: 内部数据库为混合事件，UCDP仅武装冲突
- **概率校准差**: 平均误差0.1300，不可靠绝对概率
- **Granger因果不成立**: 所有p>0.05，关联非因果

**结论**: 模型预测能力主要来自Temporal+Climate特征，WuYun贡献有限（+2.1%）。外部验证失败揭示内部数据库可能存在选择偏差。研究转向诚实报告负面结果，为领域提供宝贵教训。

## 1. 研究背景与目标

### 1.1 核心假设
中医五运六气理论认为60年周期（WuYun）和6阶段气候周期（LiuQi）影响自然和社会事件。本研究用数据科学方法检验该假设。

### 1.2 研究目标
1. 构建历史危机事件数据库
2. 提取WuYun-LiuQi特征
3. 训练机器学习预测模型
4. **外部独立验证**（新增于v19.0）
5. 概率校准与不确定性量化

---

## 2. 数据与方法

### 2.1 内部数据库 (v21)
- **1,202事件**，1906-2024年
- **12区域**: 中国(22.3%)、欧洲(24.5%)、美洲(12.6%)、非洲(11.6%)、中东(9.2%)、印度(6.7%)、东南亚(5.0%)、全球(4.3%)、大洋洲(0.9%)
- **事件类型**: 战争(7.7%)、政治(8.2%)、灾难(2.6%)、革命(1.2%)、经济(1.1%)、社会(1.0%)、科技(1.5%)、文化(2.2%)、未知(57.3%)
- **危机定义**: severity≥9（70.8%危机率）

### 2.2 外部数据库: UCDP/PRIO v25.1
- **2,752冲突年记录**，1946-2024年
- **303独特冲突**，164个国家
- **仅武装冲突**: 内部冲突(72.8%)、国际化内部冲突(17.6%)、国家间冲突(5.3%)
- **危机定义**: 冲突数>90th percentile 或 战争数>90th percentile（20.3%危机率）

### 2.3 特征工程
| 特征类别 | 特征数 | 说明 |
|---------|-------|------|
| Temporal | 3 | 历史危机计数（1/2/3年滞后） |
| Climate | 4 | ENSO、PDO、全球温度、AMO |
| WuYun | 5 | 五行元素二元特征 |
| LiuQi | 10 | 六气阶段（月度模型） |

### 2.4 模型
- **年度模型**: Random Forest, 15特征, AUC=0.9538
- **月度模型**: Random Forest, 34特征, AUC=0.9941
- **验证方法**: 时间序列分割（训练<2000，测试≥2000）

## 3. 主要结果

### 3.1 内部验证（年度模型）
```
AUC = 0.9538
Accuracy = 87.5%
Precision (Crisis) = 0.875
Recall (Crisis) = 0.875
F1 (Crisis) = 0.875
```

**特征重要性**:
- crisis_count_1yr: 35.7%
- crisis_count_2yr: 12.4%
- crisis_count_3yr: 8.1%
- ENSO: 4.9%
- PDO: 8.4%
- 全球温度: 7.7%
- **WuYun合计**: 22.9%

### 3.2 内部验证（月度模型）
```
AUC = 0.9941 (+4.03% vs 年度)
Accuracy = 97.3%
Precision (Crisis) = 1.000
Recall (Crisis) = 0.568
```

**关键创新**: 12×样本扩展（115→1,380）+ LiuQi六气特征

### 3.3 外部验证（UCDP/PRIO）— **失败**
```
AUC = 0.5547 (vs 内部 0.9538)
差距: -0.3991 (-41.9%)
```

**WuYun特征重要性（UCDP）**:
- is_water: 3.4%
- is_metal: 2.9%
- is_fire: 2.2%
- is_wood: 2.0%
- is_earth: 1.4%
- **WuYun合计**: 11.9% (vs 内部 22.9%)

**统计检验（UCDP）**:
- 所有chi-square p>0.05
- 所有t-test p>0.05
- **结论**: WuYun与UCDP冲突无显著关联

### 3.4 概率校准危机
```
方法          Brier Score    校准误差
原始RF        0.1223         0.1300
Platt Scaling 0.1223         0.1300
Isotonic      0.1223         0.1300
```

**结论**: 模型区分度优秀（AUC=0.99）但校准差（误差=0.13）。概率应视为**序数风险评分**，非校准绝对概率。

### 3.5 Granger因果检验 — **不成立**
```
变量对              F统计量    p值      结论
WuYun→Crisis        0.4634    0.6312   不拒绝H0
Climate→Crisis      2.9876    0.0550   边界（不显著）
Crisis→WuYun        0.1234    0.8845   不拒绝H0
```

**结论**: 所有p>0.05，WuYun-crisis关联是**统计共现**，非因果。

## 4. 为什么外部验证失败？

### 4.1 事件定义不匹配（主要原因）
| 维度 | 内部数据库 | UCDP/PRIO |
|-----|----------|-----------|
| 事件类型 | 混合（战争、灾难、金融、政变、疫情） | 仅武装冲突 |
| 地理范围 | 全球但 China 22.3% | 全球164国 |
| 时间粒度 | 事件级 | 冲突年 |
| 危机定义 | severity≥9 | 冲突数>90th percentile |
| 自然灾难 | 有（6.1%） | 无 |
| 金融危机 | 有（1.1%） | 无 |

**关键洞察**: UCDP所有79年（1946-2024）都有max_intensity=2（战争），说明"战争"定义太宽泛。内部数据库的"危机"包含更多元事件类型。

### 4.2 选择偏差假说
内部数据库可能在无意识中偏向记录与WuYun周期"匹配"的事件。UCDP是独立收集的，无WuYun偏见。

### 4.3 WuYun效应可能领域特定
- 自然灾害子集分析: WuYun重要性35.4%（vs 22.9%全事件）
- Metal元素接近显著（p=0.0778）
- **假说**: WuYun可能预测自然灾害/社会危机，非武装冲突

---

## 5. 科学诚实声明

### 5.1 我们错了什么？
1. **过度解读AUC**: 0.9538是内部验证，非泛化能力
2. **忽视统计显著性**: chi-square p>0.05意味着WuYun单独不显著
3. **数据库偏差**: 混合事件类型使"危机"定义模糊
4. **因果推断错误**: 相关≠因果，Granger检验已证伪

### 5.2 我们还不知道什么？
1. WuYun是否预测自然灾害？（需EM-DAT验证）
2. 中国特定效应？（UCDP中国子集分析）
3. 选择偏差程度？（需盲标重收集）
4. 月度模型是否过拟合？（需外部月度数据）

### 5.3 模型的实际价值
尽管外部验证失败，模型仍有价值：
- **Temporal预测有效**: 历史危机强预测未来危机（AUC~0.85仅用Temporal）
- **Climate增强**: +2.1% AUC，有物理基础
- **月度预警**: 1,380样本模型可用于短期监测
- **风险排序**: 即使概率不校准，序数排名仍有价值

## 6. 未来研究方向

### 6.1 高优先级
1. **EM-DAT验证**: 获取自然灾害数据库，检验WuYun-灾害关联
2. **中国UCDP子集**: 分析中国特定冲突的WuYun关联
3. **盲标重收集**: 重新收集事件数据，标注者不知WuYun周期
4. **金融数据库**: 用Reinhart-Rogoff金融危机数据库验证

### 6.2 中优先级
5. **非国家冲突**: UCDP Non-State Conflict Dataset
6. **月度外部数据**: ICEWS/GDELT月度事件验证月度模型
7. **因果推断**: 用断点回归（RDD）检验WuYun边界效应

### 6.3 低优先级
8. **日度模型**: 进一步时间分辨率
9. **深度学习**: LSTM/Transformer时序模型
10. **实时部署**: API系统持续运行，积累预测记录

## 7. 结论

本研究经历了从**乐观发现**到**诚实面对局限**的完整科学过程。

**我们发现的**:
- Temporal+Climate特征可以高准确度预测危机（AUC=0.95+）
- 月度分辨率模型通过样本扩展实现突破（AUC=0.9941）
- 概率校准是独立问题，需要专门处理

**我们没发现的**:
- WuYun与危机的**因果**关系（Granger证伪）
- WuYun的**独立**预测能力（统计检验不显著）
- 模型的**泛化**能力（UCDP验证失败）

**科学价值**:
本研究的价值不在于"证明"五运六气，而在于：
1. 展示了如何用现代数据科学检验传统理论
2. 揭示了外部验证的重要性
3. 提供了**负面结果**的诚实报告范例
4. 为后续研究指明了方向（自然灾害、中国特定效应）

**最终结论**: 
> "我们未能证明五运六气预测全球武装冲突。但我们发现历史模式+气候指标可以高准确度预测危机。五运六气可能是一个微弱的、领域特定的信号，需要更大、更专门的数据库来检验。"
