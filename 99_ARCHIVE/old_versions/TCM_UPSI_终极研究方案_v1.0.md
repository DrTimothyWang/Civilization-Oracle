核心命题:
=========
人类历史不是随机事件的堆砌，而是多层周期（气候周期、五运六气周期、精英心理周期、
社会网络共振）叠加的复杂系统。中医时间医学（五运六气、二十四节气、甲子周期）
可能编码了这些周期的经验规律。我们用AI挖掘这些隐藏的因果和逻辑关系。

"蒲公英"隐喻:
=============
每个历史精英是一个蒲公英：
- 根 = 个人命理/八字/紫微 (先天禀赋)
- 茎 = 社会网络位置 (CBDB关系数据)
- 叶 = 知识/思想产出 (文本语义)
- 花 = 权力/影响力峰值
- 种子 = 决策/政策/行动，随风飘散 (社会传播)
- 风 = 气候周期 + 五运六气 + 大历史趋势

种子落在其他节点生根 → 历史走向改变
多个蒲公英同时释放种子 → 社会相变 (革命/崩溃/复兴)

研究架构:
=========
TCM-UPSI 五层模型:

Layer 1: 宇宙-气候层 (Cosmic-Climatic Layer)
├── 米兰科维奇周期 (轨道偏心率、黄赤交角、岁差) → 10万年/4万年/2.3万年
├── 太阳黑子周期 (Schwabe 11年, Gleissberg 80-90年, Suess 200-210年)
├── 竺可桢曲线 (中国5000年温度重建)
├── 五运六气 (60年甲子周期, 10年大运, 1年主气客气)
├── 二十四节气 (太阳黄经15°划分, 年度农业-社会节律)
└── 输出: 气候压力指数 (CPI, Climatic Pressure Index)

Layer 2: 个体-心理层 (Individual-Psychological Layer)
├── 八字排盘 (出生年月日时 → 五行强弱 → 性格/决策模式)
├── 紫微斗数 (12宫 → 事业/财富/人际/健康倾向)
├── 文本语义分析 (CBDB传记 → 心理状态时序)
├── 社会网络中心性 (PageRank, betweenness)
└── 输出: 精英心理状态指数 (EPI, Elite Psychological Index)

Layer 3: 网络-传播层 (Network-Propagation Layer)
├── 蒲公英模型 (Dandelion Model)
│   ├── 节点 = 精英个体 (属性: EPI, 命理, 网络位置)
│   ├── 边 = 社会关系 (师生、同僚、亲属、政敌)
│   ├── 种子传播 = 思想/政策/行动的社会扩散
│   └── 风场 = 气候压力 + 政治压力 + 经济压力
├── 级联失效模型 (Cascading Failure)
├── 意见动力学 (Opinion Dynamics, DeGroot / Hegselmann-Krause)
└── 输出: 社会网络压力指数 (NPI, Network Pressure Index)

Layer 4: 事件-结构层 (Event-Structural Layer)
├── UPSI v17 (Material + Fragmentation + Disengagement)
├── SPI burst detection
├── 王朝周期律 (建立→兴盛→衰退→崩溃→重建)
├── 战争/革命/金融危机数据库
└── 输出: 统一压力同步指数 (UPSI)

Layer 5: 预测-干预层 (Prediction-Intervention Layer)
├── 因果推断 (DoWhy / CausalML)
├── 反事实模拟 (Counterfactual Simulation)
├── 政策干预效果评估
└── 输出: 可操作的预警与政策建议

关键科学问题:
=============
Q1: 五运六气的60年周期是否与王朝兴衰周期存在统计关联?
Q2: 精英八字中的"伤官见官""枭神夺食"等格局是否与其政治命运相关?
Q3: 气候变冷期(竺可桢曲线)是否系统性地增加社会崩溃概率?
Q4: 蒲公英网络中的"超级传播者"(高PageRank精英)是否主导历史走向?

Q6: 多层周期(气候+五运六气+个体命理)的共振是否预测重大历史事件?

1. 气候数据:

方法论:
=======
1. 周期检测:
   - Lomb-Scargle周期图 (非均匀采样时间序列)
   - 小波分析 (Morlet小波, 多尺度周期)
   - 经验模态分解 (EMD, 自适应分解)
   - 奇异谱分析 (SSA, 噪声鲁棒)

2. 因果推断:
   - Granger因果检验 (时间优先性)
   - 收敛交叉映射 (CCM, 非线性因果)
   - 因果发现算法 (PC, FCI, NOTEARS)
   - DoWhy框架 (因果图 + 估计 + 反驳)

3. 网络分析:
   - 复杂网络指标 (度分布、聚类系数、平均路径)
   - 社区检测 (Louvain, Leiden)
   - 级联模拟 (Independent Cascade, Linear Threshold)
   - 网络韧性 (渗流理论, 攻击/故障容忍)

4. 机器学习:
   - 图神经网络 (GNN, 社会网络预测)
   - 时序模型 (LSTM, Transformer, N-BEATS)
   - 因果表示学习 (因果VAE, 因果GAN)
   - 强化学习 (政策优化)

5. 中医特定方法:
   - 五运六气计算引擎 (天干地支 → 五行生克)
   - 八字格局自动识别 (正官格、七杀格、伤官格等)
   - 紫微斗数星曜组合分析
   - 节气-物候-社会事件关联挖掘
