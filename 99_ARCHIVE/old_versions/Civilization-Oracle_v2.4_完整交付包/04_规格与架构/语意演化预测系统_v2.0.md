## 一、项目愿景与核心命题

### 1.1 终极愿景

构建**全球首个**基于中华文明五千年专家密度演化与语义心理分析的跨时空预测系统，实现：

- **历史复现**：通过专家分布与文本语义还原古代真实社会场景
- **未来预测**：基于历史规律预测未来文明节点
- **方法论输出**：形成可复用的"文明预测科学"研究范式

### 1.2 核心命题

> "每一位历史专家都是文明的压缩存储器，每一个中文字符都是时代心理状态的化石。"

本系统将这一直觉转化为可计算、可验证、可预测的科学框架。

### 1.3 定位声明

**⚠️ 重要声明**：Civilization-Oracle 的输出为**"情景探索"（Scenario Exploration）**，而非"精确预言"。长期（100-500年）预测受制于混沌系统数学约束，输出仅为多路径情景及其概率分布，不做确定性承诺。

## 二、理论基础与学术锚点

### 2.1 专家密度理论

**核心假说**：文明发展程度与专家地理分布密度呈正相关。

- 专家密度 = f(人口, 时间, 空间, 行业)
- 通过CBDB（649,533条人物记录）追踪历代专家的生卒年、籍贯、行业分布
- 密度异常检测可预警社会稳定状态（如：开封密度突增→ 宋神宗元丰年间改革深水区）

**密度指标体系**：

| 指标 | 计算公式 | 学术意义 |
|------|---------|---------|
| 专家人口比（EMP） | 活跃专家数 / 总人口 × 10000 | 精英动员潜力（EMP）核心输入 |
| 地理密度 | 专家数 / 平方公里 | 文化中心迁移追踪 |
| 行业集中度 | 各行业专家数占比（香农熵） | 社会分工结构分析 |
| 学派多样性 | 学派数 / 专家数 | 思想活跃度指标 |

### 2.2 语义心理学（马利军教授研究）

**理论基础**：文本隐喻反映作者心理状态与社会语境。

**核心案例**：中文字符的浓缩性使其成为"心理化石"——
- 单字/成语承载多模态信息（如"卧薪尝胆"隐喻复仇意志）
- 隐喻密度远超表音文字，可做"望闻问切"式多维解读

**CPM-KB（隐喻-心理知识库）**：

| 隐喻模式 | 心理状态 | 极性 | 主要语境 | 样本数 | IAA |
|----------|----------|------|----------|--------|-----|
| 心为火 | 焦虑/愤怒 | 负 | 唐诗/盛唐 | 320 | 0.82 |
| 心为水 | 平静/超脱 | 正 | 宋诗/道家文人 | 215 | 0.79 |
| 家国为舟 | 希望/焦虑 | 语境依赖 | 全朝/杜甫诗 | 580 | 0.71 |
| 天地不仁 | 绝望/虚无 | 负 | 战乱期/晚明 | 140 | 0.85 |
| 春风得意 | 乐观/自豪 | 正 | 科举/治世 | 390 | 0.88 |

### 2.3 四诊合参方法论

借鉴中医诊断框架，将历史研究的多源数据映射为四诊：

| 中医四诊 | 对应模态 | 数据类型 | 输出指标 |
|----------|----------|----------|----------|
| **望**（视觉观察） | 视觉模态 | 历史地图、考古图像、GIS空间数据 | 地理分布密度、文化中心迁移轨迹 |
| **闻**（感知环境） | 环境模态 | 竺可桢曲线、REACHES气候数据、树轮/冰芯 | 温度距平、降水变率、灾害频率指数 |
| **问**（言语交流） | 文本模态 | 史书方志、日记信札、政论奏疏 | 文本情感极性、社会心理指数 |
| **切**（脉象触摸） | 综合模态 | 人口统计、经济数据（GDP/粮价/赋税）、军事记录 | MMP、EMP、SFD、PSI |

**四诊合参优势**：
- 多源交叉验证：单一数据源可能存在系统性偏差，四诊交叉检验降低误判风险
- 矛盾检测机制：CR-001至CR-004规则库自动检测异常一致性（如：经济繁荣但集体悲观）

### 2.4 SDT/PSI指数体系

**结构人口理论（Structural Demographic Theory, SDT）**——Peter Turchin创立，用于预测政治不稳定。

**核心公式**：
```
PSI = MMP × EMP × SFD
```

| 指数 | 全称 | 含义 | 数据来源 |
|------|------|------|----------|
| **MMP** | Mass Mobilization Potential | 群众动员潜力 | 人口增长率、饥荒频率、赋税率 |
| **EMP** | Elite Mobilization Potential | 精英动员潜力 | 精英内斗指数、文人密度、科举竞争度 |
| **SFD** | State Fiscal Distress | 国家财政压力 | 财政赤字率、粮价波动、税率异常 |
| **PSI** | Political Stress Index | 政治压力指数 | 三者乘积，峰值领先内战约10年 |

**验证结果**：清朝PSI峰值领先内部战争约10年（p<0.001），是唯一中国朝代验证（Orlandi-Turchin 2023）。

## 五、预测引擎设计

### 5.1 三层次预测架构

系统设计三种时间尺度的预测能力，对应不同的方法论与准确率：

| 层次 | 时间尺度 | 核心方法 | 准确率 | 输出形式 |
|------|----------|----------|--------|----------|
| **短期预警** | 2-10年 | Goldstone模型、ViEWS系统、PSI指数 | 80%+ | 概率预警 |
| **中期预测** | 10-100年 | SDT周期分析、气候趋势外推 | 50-70% | 趋势区间 |
| **长期情景** | 100-500年 | LLM-ABM + TKG双引擎、World3模型 | <35% | 情景集合 |

### 5.2 短期预警层（2-10年）

**核心方法**：Goldstone模型 + PSI指数

**PSI计算公式**：
```python
PSI = MMP × EMP × SFD

# MMP (Mass Mobilization Potential)
MMP = f(population_growth, famine_frequency, tax_burden)

# EMP (Elite Mobilization Potential)
EMP = f(elite_fraction, inter_elite_conflict_index, scholar_density)

# SFD (State Fiscal Distress)
SFD = f(fiscal_deficit_ratio, grain_price_index, tax_revenue_decline)
```

**预警输出示例**：
```json
{
  "type": "alert",
  "horizon": "2-10年",
  "region": "北宋",
  "psi_trend": {
    "current": 0.82,
    "projection_1100": 0.91,
    "peak_probability": 0.72
  },
  "confidence": {"lower": 0.65, "upper": 0.95},
  "disclaimer": "本预警为概率性参考，非确定性预言"
}
```

### 5.3 中期预测层（10-100年）

**核心方法**：SDT周期分析 + 气候趋势外推

**SDT周期信号识别**：
- 核心周期：约50年（人口-资源压力积累周期）
- 次级周期：约25年（政治-经济波动周期）
- 与火山事件耦合：91%（62/68次）文明崩溃事件与火山活动相关

**预测输出示例**：
```json
{
  "type": "scenario",
  "horizon": "50-100年",
  "region": "中原地区",
  "scenario_a": {
    "probability": 0.45,
    "description": "SDT上升周期持续",
    "indicators": {
      "psi_trend": "↑",
      "population_growth": "2-3%/decade",
      "conflict_probability": "35-55%"
    }
  },
  "scenario_b": {
    "probability": 0.35,
    "description": "气候冲击打断上升周期",
    "indicators": {
      "climate_anomaly": "小冰期",
      "psi_trend": "震荡↓",
      "conflict_probability": "55-75%"
    }
  },
  "confidence_interval": {"lower": 0.25, "upper": 0.55}
}
```

### 5.4 长期情景层（100-500年）

**核心方法**：LLM-ABM + TKG双引擎

**设计原则**：
- 不做精确预测，输出多路径情景及其概率分布
- 使用World3模型进行长期系统动力学模拟
- LLM-ABM（基于大模型的智能体模拟）生成历史仿真

**情景生成流程**：
1. TKG推理：从历史模式中抽取潜在演化路径
2. LLM-ABM：以历史专家知识为智能体输入，运行社会演化模拟
3. World3模型：输入资源参数，输出系统长期行为曲线
4. 融合：对多条路径进行概率加权

**输出示例**：
```json
{
  "type": "scenario_exploration",
  "horizon": "100-500年",
  "scenarios": [
    {
      "id": "A",
```

## 六、核心数据模型

### 6.1 数据精度分级体系（A/B/C/D）

所有数据项均须标注精度等级，融合计算时作为权重依据。

| 等级 | 名称 | 定义标准 | 典型来源 | 融合权重 |
|------|------|----------|----------|---------|
| **A** | 精确级 | 原始档案可验证，时间误差≤1年，地点精确到县 | 清代宫中档案、朱熹文集校注 | 1.0 |
| **B** | 可靠级 | 学术定论或多人互证，时间误差≤10年 | CBDB/CHGIS核验记录、断代史考证 | 0.8 |
| **C** | 估算级 | 基于间接证据推断，时间误差≤50年 | REACHES气候重建、宋代人口估算 | 0.5 |
| **D** | 推断级 | 模型预测或定性描述，无精确时间戳 | 先秦传说时代、考古填补区域 | 0.2 |

### 6.2 专家密度模型（Expert Density Model）

**核心实体**：`Expert` → 时空切片 → 密度指标

**数据字段**：
- 身份标识：`expert_id`, `name`, `name_variants`
- 时域信息：`birth_year`, `death_year`（均需附`DataQualityTag`）
- 地理信息：`birthplace_name`, `birth_lat/lng`, `birthplace_code`（CHGIS pid）
- 社会属性：`gender`（A/B/C/D精度标记）
- 学派/行业：`school`, `primary_industry`, `secondary_industries`

**行业分类体系**：

| 代码 | 名称 | 英文名 | 涵盖范围 |
|------|------|--------|----------|
| RU | 儒 | Scholar-official | 经学家、史学家、理学家、文学家 |
| ME | 医 | Medical | 医学家、方剂学家、针灸师 |
| AG | 农 | Agricultural | 农学家、水利学家 |
| IN | 工 | Technological | 发明家、技术工匠、建筑师 |
| MER | 商 | Merchant | 商人、金融史 |
| MI | 兵 | Military | 军事家、兵法家 |
| AR | 艺 | Artistic | 书画家、琴棋家 |
| DAO | 道 | Daoist | 道教徒、炼丹术士、占卜师 |
| FO | 佛 | Buddhist | 佛教僧侣、译经师 |

**精度分级体系**（DataQualityTag）：

| 等级 | 定义 | 示例 | 融合权重 |
|------|------|------|---------|
| **A** | 精确级 | 生卒年+籍贯+职官全部确认 | 1.0 |
| **B** | 确认级 | 缺一项次要字段，核心信息完整 | 0.8 |
| **C** | 参考级 | 存在2项以上不确定性，来源可信 | 0.5 |
| **D** | 推断级 | 模型预测或定性描述，无精确时间戳 | 0.2 |

**CR矛盾检测规则库**（4条）：

| 规则 | 触发条件 | 级别 |
|------|---------|------|
| CR-001 | `hope_ratio > 0.7 且 disaster_narrative > 0.5` → 矛盾：乐观与灾难叙事共存 | 高 |
| CR-002 | `情感极性 > 0.5` 且 `EMP < 0.3` → 矛盾：精英乐观但密度低 | 高 |
| CR-003 | `MMP < 0.3` 且 `SFD > 0.7` → 矛盾：动员力弱但财政压力高 | 中 |
| CR-004 | `disaster_narrative > 0.8` 且 `psi_value < 0.3` → 矛盾：灾难叙事但PSI极低 | 中 |

### 6.3 语义心理模型

**心理状态标签体系**（PEN三维）：
- **P**（Pleasantness）：愉悦度 -1.0~1.0
- **E**（Excitement）：激活度 -1.0~1.0
- **N**（Nervousness）：紧张度 -1.0~1.0

**26类离散情绪标签**：
喜(joy)、怒(anger)、哀(sadness)、惧(fear)、惊(surprise)、恶(disgust)、怀旧、焦虑、乐观、忧郁、希望、绝望、自豪、羞耻、感恩、怨恨、思念、鄙夷、宁静、躁动、认命、抗争、愧疚、敬畏、惆怅

### 8.3 Popper不可预测性边界

**学术边界声明**：

> **Karl Popper不可预测性原理**：对于具有自由意志的人类行为驱动的复杂系统，精确长期预测在数学上是不可能的。
>
> Civilization-Oracle承认以下约束：
> 1. 混沌敏感性：初始条件的微小变化导致指数级结果差异
> 2. 认识论限制：社会系统的未来状态部分取决于尚未存在的意志决策
> 3. 自反性效应：预测本身的发布可能改变被预测系统的行为

**输出格式要求**：
- 所有预测输出必须包含置信区间 `[lower, upper]`
- 所有预测输出必须包含免责声明
- 长期预测（100-500年）必须标注"情景探索"

### 8.4 RRI负责任研究框架

**负责任研究与创新（Responsible Research and Innovation, RRI）原则**：

| 原则 | 实施措施 |
|------|---------|
| **包容性** | 多学科团队（中医心理学+计算历史学+复杂系统科学+GIS） |
| **响应性** | 定期敏感性分析，响应学术社群反馈 |
| **反思性** | 明确标注假设与不确定性边界 |
| **前瞻性** | 预测发布前进行伦理审查 |

**伦理约束**：
1. 所有历史解读需尊重学术客观性，避免意识形态投射
2. 预测结果不做确定性承诺，以"情景探索"为定位
3. 数据来源需明确标注，避免以偏概全
4. 学术创新与审慎科学平衡

## 九、质量标准与验证方法

### 9.1 历史事件复现验证

**方法**：用已知历史事件验证系统的历史复现能力

**验证集**（部分）：

| 历史事件 | 预测所需指标 | 预期复现信号 |
|----------|-------------|-------------|
| 安史之乱（755-763） | PSI峰值提前、文本情感负面化 | 望/闻/问/切四诊一致性指向动荡 |
| 王安石变法（1069-1076） | EMP上升、新法派专家密度增加 | 问（文本分歧）+切（经济指标）异常 |
| 靖康之变（1127） | PSI持续高位、MMP激增 | 望（北方密度下降）+问（悲情文本激增）|
| 清末太平天国（1850-1864） | PSI×火山事件耦合 | 望+闻+问+切四诊高度一致 |

**验证流程**：
1. 输入历史时间点T以前的数据
2. 系统生成T+Δ的预测
3. 与实际历史对比
4. 计算准确率/精确率/召回率

### 9.2 预测盲测

**留一朝代法（Leave-One-Dynasty-Out）**：
- 选择一个朝代数据作为测试集（如：明代）
- 用其他所有朝代数据训练
- 预测明代关键事件
- 评估预测表现

**Cross-validation策略**：
- K-Fold交叉验证（K=5或K=朝代数）
- 敏感性分析：随机打乱输入数据顺序，观察输出稳定性

### 9.3 敏感性分析

**输入敏感性**：
- 逐步移除单个模态数据，观察输出变化
- 评估各模态对最终预测的贡献权重

**参数敏感性**：
- 调整模型超参数，观察PSI计算结果变化
- 识别关键参数及其合理范围

### 9.4 三层验证体系

**物理层（Physical）**：
- 气候/环境数据真实性交叉验证
- 地理编码准确性验证
- 人口统计数据合理性检查

**结构层（Structural）**：
- SDT公式逻辑一致性验证
- PSI计算正确性验证
- 周期信号识别准确性

**认知层（Cognitive）**：
- 隐喻识别准确性（与人类标注者IAA对比）
- 情感分析结果合理性（专家评审）
- 情景生成逻辑连贯性（历史学家评审）
