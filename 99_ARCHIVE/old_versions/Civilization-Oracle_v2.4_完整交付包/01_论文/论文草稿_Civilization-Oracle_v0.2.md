## 2. 方法论

### 2.1 贝叶斯层次推断框架

**问题诊断**：OLS回归对北宋7周期数据给出R²=0.68，表面看起来不错。但经自由度校正后，**Adjusted R²仅为0.36**，说明高估了模型的解释力。这在小样本(n=7, k=3)情况下是典型的过拟合陷阱。

**解决方案**：
1. **Bootstrap层次回归**：2000次重抽样，估计系数分布，避免小样本置信区间过窄
2. **Jeffreys无信息先验**：对PSI的后验分布不引入主观假设
3. **Hedges' g 小样本修正**：Cohen's d 的小样本偏差修正常见于心理学和医学研究

**验证结果**（北宋7周期）：
| 指标 | 原始值 | 贝叶斯校正 |
|------|--------|-----------|
| PSI均值 | 0.6309 | 0.6309 |
| PSI标准差 | 未报告 | 0.0584 |
| 95%CI | 未报告 | (0.5027, 0.7313) |
| Adjusted R² | 0.36 | 0.36 (校正后) |

### 2.2 SikuBERT 古籍NLP

**问题诊断**：通用BERT（如BERT-base-Chinese）基于现代白话文训练，无法捕捉：
- 文言文专有词汇（如"积弊""党争"）
- 隐喻表达（"身体_为_朝廷"）
- 历史语境依赖性（同一词在不同时期情感不同）

**解决方案**（v2.4降级方案）：
1. 内嵌20个北宋高频概念词典（政治/社会/经济/文化/军事五域）
2. 规则匹配 + 情感值加权
3. 完整SikuBERT模型可在后续接入（需 `pip install transformers`）

**验证结果**：

|----------|----------|------|
| "嘉祐之治，天下太平，盛世繁华" | +0.800 | 正面 ✓ |

### 2.3 CHGIS 地理编码

**问题诊断**：CBDB中北方专家占比过高（开封、洛阳周边），导致PSI系统性高估。需要空间校正。

**解决方案**：
1. **秦岭-淮河分界线**（34°N）：区分北方政治核心区 vs 南方经济文化区
2. **地理压力指数（GSI）**：整合区域分布 + 专家密度
3. **TSI地缘压力推理**：north_ratio > 0.20 → 触发危机预警

**参数配置**：
```
北方压力系数: 1.4x
南方压力系数: 0.8x
边疆压力系数: 1.2x
危机阈值（north_ratio）: > 0.20
```

**验证结果**（5位北宋专家，3北2南）：
- GSI = 1.3678（偏高，因为北方占多数）
- north_ratio = 0.6
- stability = "crisis"（触发危机预警 ✓）
