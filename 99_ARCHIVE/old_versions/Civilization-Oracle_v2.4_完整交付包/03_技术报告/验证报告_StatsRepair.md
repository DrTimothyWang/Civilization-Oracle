## 1. Adjusted R² 计算验证

### 北宋原始回归

| 参数 | 值 |
|------|-----|
| 原始 R² | 0.68 |
| 样本量 n | 7 |
| 自变量数 k | 3 |

### 计算过程

```
Adjusted R² = 1 - (1 - R²) × (n - 1) / (n - k - 1)
           = 1 - (1 - 0.68) × (7 - 1) / (7 - 3 - 1)
           = 1 - 0.32 × 6 / 3
           = 1 - 0.32 × 2
           = 1 - 0.64
           = 0.36
```

### 验证结果

✅ **通过** — Adjusted R² = 0.36，与预期完全一致。

> 注：从 R²=0.68 降至 Adjusted R²=0.36，惩罚量达 0.32，表明在 n=7 的小样本下，k=3 个自变量对模型过拟合有显著影响。

## 3. Holm 校正验证（k=10 次检验）

### 模拟 p 值序列

假设进行了 10 次统计检验，原始 p 值如下：

| 检验 # | 原始 p 值 | 排序 |
|--------|----------|------|
| H1 | 0.001 | 1 |
| H2 | 0.005 | 2 |
| H3 | 0.012 | 3 |
| H4 | 0.020 | 4 |
| H5 | 0.035 | 5 |
| H6 | 0.048 | 6 |
| H7 | 0.062 | 7 |
| H8 | 0.080 | 8 |
| H9 | 0.120 | 9 |
| H10 | 0.250 | 10 |

### Holm 校正阈值（α=0.05）

| 步骤 i | 阈值 α/(k-i+1) |
|--------|----------------|
| 1 | 0.05/10 = 0.0050 |
| 2 | 0.05/9  = 0.0056 |
| 3 | 0.05/8  = 0.0063 |
| 4 | 0.05/7  = 0.0071 |
| 5 | 0.05/6  = 0.0083 |
| 6 | 0.05/5  = 0.0100 |
| 7 | 0.05/4  = 0.0125 |
| 8 | 0.05/3  = 0.0167 |
| 9 | 0.05/2  = 0.0250 |
| 10 | 0.05/1  = 0.0500 |

### 逐步判断

- **i=1**: p(1)=0.001 ≤ 0.005 → **通过**，停止（step-down 程序）

→ H1 被拒绝；H2-H10 均标记为校正后的拒绝

### 校正结果

| 检验 # | 原始 p | Holm 校正后 p | 结论 (α=0.05) |
|--------|--------|--------------|-------------|
| H1 | 0.001 | min(0.001×10, 1)=0.01 | ✅ 拒绝 H₀ |
| H2 | 0.005 | min(0.005×9, 1)=0.045 | ✅ 拒绝 H₀ |
| H3 | 0.012 | min(0.012×8, 1)=0.096 | ❌ 不显著 |
| H4–H10 | … | … | ❌ 不显著 |

> 结论：经过 Holm-Bonferroni 校正后，仅 H1（p=0.001）和 H2（p=0.005）在 α=0.05 水平下显著。相比原始 Bonferroni 校正（阈值 0.005），Holm 方法在 H2 处保留了显著性，功效更高。
