## 一、执行摘要

Civilization-Oracle 是一个基于中华文明五千年专家密度与语义心理分析的历史预测系统，旨在通过"四诊合参"多模态框架探索文明演化的深层语义驱动力。项目历经 P3 至 P7 五个迭代里程碑（2026-05-28 至 2026-05-30），完成了从"单周期北宋"到"全历史覆盖 + 云端 NLP + 标准协议栈"的三维跨越。

核心成果包括：PSI（心理状态指数）覆盖五朝 30,518 名历史人物，唐朝 PSI=0.6122（开元盛世）、明朝 PSI=0.6250（永乐盛世）、南宋 PSI=0.3804（偏安终局），与历史事实高度吻合；TKG v3.0 融合算法 MRR 达 36.31%，较基线提升 22.6%；MCP+A2A 标准协议栈实现了 8 个 Agent 的标准化互操作；四诊合参 2.0 一致性达 0.8407，通过 0.7 阈值验收。论文草稿 v0.5 达 33,332 字符，目标期刊 Digital Humanities Quarterly（DHQ），拟于 2026-07-15 投稿。

## 二、研究背景与目标

### 2.1 科学问题

历史文明演化是否存在可辨识的语义压力先行指标？Peter Turchin 的 Cliodynamics（历史动力学）提出历史可预测假设，但其模型缺乏语义心理维度。Civilization-Oracle 在此基础上提出核心假设：**精英群体的语义心理状态密度（PSI）是文明危机的有效先行指标，峰值领先内战约 10 年**。

### 2.2 理论框架

项目整合三大理论来源：

- **SDT（信号检测理论）**：区分真实信号（文明危机语义）与噪声（偶发性文本情绪），以 d' 指标量化 PSI 的判别能力；
- **PSI 公式**：PSI = (0.25×MMP + 0.25×EMP + 0.5×SFD) × GSI，其中 MMP 为情感极性均值、EMP 为情绪比值、SFD 为专家密度标准化值、GSI 为地理稳定性指数；
- **四诊合参**：借鉴中医诊断思想，引入望（地理）、闻（气候）、问（语义）、切（综合 PSI）四维交叉验证机制，提升结果可信度。

### 2.3 研究目标

| 阶段 | 目标 | 状态 |
|------|------|------|
| v2.4 | 单周期北宋 PSI 基线验证 | ✅ |

| v2.6 | 统计敏感性分析 + 论文冲刺 | ✅ |

### 3.2 五大模块

**模块一：数据工程（Track 2）**

- 数据质量分级：A 级（生卒年+姓名+籍贯，score≥7）、B 级（有生年+姓名，score≥4）

**模块二：NLP 与语义分析（Track 3）**
- 云端 API：MiniMax-M2.7-highspeed（Base URL: `api.minimaxi.com/v1`），支持自动降级至通义/Claude

- 隐喻知识库：CPM-KB 从 10 条扩展至 1,000 条（TSI 半自动框架）

**模块三：TKG 时序知识图谱（Track 3）**
- TKG v3.0 融合架构：DiMNet（45%）+ TransFIR（40%）+ TGL-LLM（15%）
- MRR = 0.3631，较基线（v2.6 TKG-LDG MRR=0.2963）提升 22.6%

**模块四：四诊合参 2.0（Track 2 + P7）**
- 望：CNHGIS 地理编码，内置 15 个古今地名映射，AI 动态推断未知地名
- 闻：竺可桢气候温度曲线 → 温度距平 + 灾害频率
- 问：MiniMax API 文本语义分析 → 情感极性 MMP
- 切：综合 PSI（望+问加权融合），一致性阈值 0.7

**模块五：MCP+A2A 协议栈（P7）**
- 8 个 Agent Card 动态发现（`.well-known/agent.json`）
- 5 个 MCP Tool：psi_analyze、cbdb_query、tkgr_predict、quality_audit、ipw_correct
- 9 状态任务生命周期 + 迭代闭环反馈

## 八、局限性

### 8.1 样本量约束

当前 PSI 验证基于 n=7 个历史周期，Green 法则最低要求 N≥31（差距 4.4 倍），Harrell 法则要求 N≥60（差距 8.6 倍）。Adjusted R²=0.36 说明模型在考虑自由度惩罚后仅解释了约 36% 的方差。这一约束在 Track 1 贝叶斯层次框架（n<30 时推荐贝叶斯方法）和 Walk-Forward 验证方案中已有系统性应对。

### 8.2 数据代表性偏差

CBDB 存在系统性偏差：精英偏向（高官传记记载更完整）、北方政治中心偏向（北方仅 0.5%，南方 4.6%）、时期集中偏向（唐宋时期占 75%+）。女性人物数据 <1%。IPW 校正部分缓解了而非彻底解决了这一问题。

### 8.3 地理数据覆盖率

约 60-70% 的专家有地理坐标（x_coord/y_coord），CNHGIS 公开 API 尚未开放，当前所有查询走 CHGIS V6 静态映射 + AI 推断降级，泉州等次要城市超出内置知识库范围。

### 8.4 认识论边界

Popper Argument II（历史唯一性）和混沌理论共同构成了长期精确预测的根本限制。项目定位为"情景探索平台"而非"预测引擎"，输出须严格遵循认识论谦逊六原则（样本透明度、假设显式化、不确定性传播等）。

### 8.5 十年级 PSI 真实 API

96 个十年窗口的 MiniMax 真实 API 调用尚未完成（预计 16-24 分钟），当前验证基于 mock 数据。
