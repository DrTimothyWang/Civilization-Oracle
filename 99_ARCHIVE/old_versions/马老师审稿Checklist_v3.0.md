### 1. PSI 公式不一致 — 最高优先级

**问题**：论文中存在三种互相矛盾的 PSI 公式描述：

| 位置 | 公式描述 |
|------|---------|
| 摘要 | `PSI = MMP×0.25 + EMP×0.25 + GSI×0.50` |
| 3.3节 | `SFD = 0.25×MMP + 0.25×EMP + 0.5×GSI`，`PSI = SFD × density_norm` |
| 附录A | `PSI_raw = SFD × density_norm`，`SFD = 0.25×MMP + 0.25×EMP + 0.5×GSI` |

**矛盾**：
- 摘要公式：PSI = MMP×0.25 + EMP×0.25 + GSI×0.50 → **不乘以 density_norm**
- 实际数据计算：PSI = (0.25×MMP + 0.25×EMP + 0.5×GSI) × density_norm
- 这两个公式**数学上不相等**

**附录A数据反验**：
```
唐朝 PSI_raw(750s) = (0.25×(-0.9167) + 0.25×EMP + 0.5×0.81) × 1.00 = 0.4167
→ 反推 EMP = 0.8667
→ SFD = 0.25×(-0.9167) + 0.25×0.8667 + 0.5×0.81 = 0.4167
```
数据内部自洽，但与摘要公式矛盾。

**修复建议**：
1. 统一全文公式为：`PSI = SFD × density_norm`，`SFD = 0.25×MMP + 0.25×EMP + 0.5×GSI`
2. 摘要改为：`PSI = SFD × density_norm，SFD = 0.25×MMP + 0.25×EMP + 0.5×GSI（v3.0）`
3. 明确 density_norm 是专家密度归一化值

### 5. Cohen's d 计算方法

**问题**：Cohen's d = 7.35（大效应）计算如下：
```
盛世组（唐/明均值=0.6186）vs 危机组（北宋后期/南宋均值=0.4083）
```

**逻辑问题**：
- Cohen's d 应该在 individual-level 数据上计算
- 当前用的是 dynasty-level 均值（n=4），不是 individual-level（n=30,518）
- 在 aggregate means 上计算 d 会严重夸大效应量（生态学谬误）
- 真实的 within-dynasty SD 远大于 dynasty-level SD，真实 d 应该小得多

**修复建议**：
- 在 30,518 条原始记录上计算 Cohen's d（标注 dynasty label，做独立样本 t 检验）
- 或者在论文中明确说明："由于 n=5 朝代的聚合数据限制，d 为近似值"
- 建议补充 individual-level 分析（每个人物一条记录）

### 7. 时间自相关未处理

**问题**：十年级 PSI 时间线（96个十年窗口）存在强时间自相关，但 Cohen's d 和 R² 等统计量均假设观测独立。

**违反的假设**：
- 朝代内各 decade 不是独立样本（前后 decade 的 PSI 高度相关）
- 自相关系数 ρ 预计 > 0.5（相邻十年属于同一朝代趋势）
- 这会使 R² 膨胀、Cohen's d 夸大

**修复建议**：
- 补充 Newey-West 异方差自相关一致估计量（HAC）
- 或使用广义最小二乘（GLS）替代 OLS
- 在 limitation 中明确承认："本分析的统计推断未处理时间自相关，真实显著性可能被高估"
