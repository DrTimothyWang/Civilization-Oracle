## 3.11 理论框架：三阶段 + 三机制

> 来自 `v4/data/theoretical_framework.json`

```
Phase 1 (T-15 to T-10) — Trigger:
  主导机制: M2 压力传导
  PSI 行为: 缓慢下降

Phase 2 (T-10 to T-5) — Critical:
  主导机制: M1 认知窗口 + M3 网络失序
  PSI 行为: 急剧下降（谷值）

Phase 3 (T-5 to T) — Crisis:
  主导机制: 物理事件链触发
  PSI 行为: 持续低位
```
