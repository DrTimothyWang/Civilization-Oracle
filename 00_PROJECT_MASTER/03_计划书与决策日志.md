## 第一部分：项目计划书

### 1.1 总体目标

**核心命题**：精英群体语义心理状态密度（PSI）是文明危机的有效先行指标。

**终极目标**：建立一个跨域（6 域）、跨时间（4200 年）、跨学科（历史学 + 金融学 + 物理）的统一复杂系统理论，提供：
1. 学术贡献：可量化指标 + 物理机制 + 跨文化普遍性
2. 技术贡献：Nature 级投稿 + 开源代码
3. 应用贡献：监管/投资/政策三层落地

**6 年路线图**：
- 2026：Nature 投稿 + 5 朝覆盖完成

- 2028：贝叶斯层次推断 + IPW 跨数据源
- 2029：实时 PSI 监控 + 政策实施
- 2030+：Seshat 互操作 + 跨平台比较研究

### 1.3 关键设计原则

1. **科学方法论优先**：盲测合规、阴/阳性都接受、不追求全阳性
2. **诚实报告边界**：v3.0 12 个审稿问题不回避，写在 limitations 显眼位置
3. **数据驱动决策**：每次方向调整都有数据/审稿反馈支撑
4. **跨学科尊重**：马利军教授的语义心理学理论是起点，不是工具
5. **公开可复现**：v4.x 有 reproduce.py、Nature 投稿要求代码开源
6. **轻资产优先**：v2.5 后切换到云端 API（MiniMax），不依赖本地大模型

### 1.4 风险与应对

| 风险 | 等级 | 应对 |
|---|---|---|
| 样本量不足（n=7 vs Green N≥31） | 高 | 贝叶斯层次 + Walk-Forward 验证 |
| 精英偏差（CBDB 70% 官员） | 高 | IPW 校正 + 四诊合参 + 诚实声明 |
| 长期混沌效应（>300 年相符率低） | 中 | 明确"情景探索"定位，不承诺精确预测 |
| CDLI 公共 API 100 条限制 | 中 | 申请账户或完整下载，标注数据局限 |

### 决策 8（2026-05-28）：v2.4 选择北宋作为首个验证周期

- **决策**：先做北宋（960-1127），不直接做全朝代
- **理由**：
  - 北宋时间跨度适中（~150 年），数据量足够
  - 有明确危机节点（靖康之变 1127）
  - CBDB 记录质量高（3,564 条 A/B 级）
- **影响**：v2.4 证明"能做"，v2.5 扩展到五朝——渐进式验证策略

### 决策 9（2026-05-28）：PSI 公式权重 0.25/0.25/0.5（SDT 论证）

- **决策**：SFD（专家密度）权重 0.5 > MMP/EMP（文本情绪）各 0.25
- **理由**：
  - 信号检测论（SDT）：客观信号 > 主观报告
  - 专家密度是客观计数，文本情感易被政治宣传扭曲
  - 唐朝/明朝多个时期 Bootstrap 验证此权重合理
- **影响**：成为整个 PSI 框架的理论基石，v4.x 跨域公式延续此原则

### 决策 10（2026-05-27）：引入 IPW 校正 CBDB 精英偏差

- **决策**：用 IPW 逆概率加权校正，不放弃 CBDB
- **理由**：
  - CBDB 70% 是官员及家属，精英偏差真实存在
  - IPW 来自流行病学（Hernán & Robins 2023），方法论成熟
  - IPW 校正后 PSI 均值上移 +0.06，方向稳定（r=1.00）
- **影响**：方法论贡献——首次应用于 CBDB 历史数据偏差校正

### 决策 11（2026-05-27）：借鉴中医"四诊合参"做交叉验证

- **决策**：望（地理）+ 闻（气候）+ 问（语义）+ 切（综合 PSI），一致性阈值 0.7
- **理由**：
  - 单一信号易被质疑，四个独立通道交叉验证
  - 中医诊断隐喻便于马利军教授（语义心理学）理解
  - 一致性 0.8407 超过 0.7 阈值——通过验证
- **影响**：项目从"单指标"升级为"多模态架构"

### 决策 12（2026-05-27 之前）：整合三大理论来源

- **决策**：SDT（信号检测论）+ PSI 公式 + 四诊合参，三者整合
- **理由**：
  - SDT 提供权重和阈值理论依据

## 第四部分：方法论教训（沉淀给后人）

### 教训 1：公式混乱是审稿杀手

v3.0 有 4-6 种公式版本，导致 Table 矛盾。v4.x 严格统一 formula.py 一个文件。

**经验**：每个项目有"唯一真相源"——一个公式、一个权重、一个阈值。

### 教训 2：聚合级别统计有生态学谬误

v3.0 的 Cohen's d=7.35 是朝代级聚合，个体级只有 0.43（IPW 校正 0.53）。两者都成立但意义不同。

**经验**：报告效应量时必须说明聚合级别——审稿人一定问。

### 教训 3：盲测合规是不可妥协的

v3.0 12 个问题中，事件泄漏是最严重的——隐藏期事件泄漏，移除后纯时序统计仍有效，说明原结果有问题。

**经验**：训练期/测试期严格分隔，盲测规则写进数据加载代码里。

### 教训 4：长短期数据需要分层结论

短期（20-60 年）相符率 70-96.6%，长期（300+ 年）急降至 21.4-50.6%。一个数字误导性极强。

**经验**：报告结论必须分层——按时间窗口、按数据稀疏度。

### 教训 5：阴性结果有独立价值

气候对照 r=0.02（独立）、贝叶斯后验 P(唐=明)=低——这些阴性结果说明 PSI 不被气候混杂，反向证明其独立性。

**经验**：不追求全阳性——诚实报告反而提升可信度。

### 教训 6：跨学科合作要尊重对方领域

马利军教授是语义心理学，PSI 必须用其理论语言描述（隐喻、心理状态、社会语境），不能只讲技术。

**经验**：写跨学科论文时，理论框架用对方领域的术语。

### 教训 7：版本压缩包要按时归档

8 个 .zip 文件（v4x → v6）每次阶段性产出都打一个，避免最终整理遗漏。

**经验**：阶段末必归档——5.8 MB/168 files 不可能临时凑。

### 教训 8：Project memory 沉淀比临时文档更可靠

PROJECT_LOG.md + 06_Agent开发指南.md + 本档案 = 三件套，给接手 Agent 完整心智模型。

**经验**：每个项目维护"主档案 + 日志 + 协作指南"三件套。
